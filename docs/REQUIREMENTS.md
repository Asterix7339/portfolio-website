# REQUIREMENTS

## 1. Project Overview

This is a personal portfolio website for Mohamed Satham.

The purpose is to:

* Show skills and projects
* Help apply for internships and jobs
* Present a professional online identity

---

## 2. Core Features

The website must include:

### 2.1 Navigation

* Fixed navigation bar
* Smooth scrolling to sections

### 2.2 Hero Section

* Name display
* Professional title
* Short introduction
* Call-to-action buttons:

  * View Projects
  * Contact Me

### 2.3 About Section

* Short biography
* Career goal
* Education summary

### 2.4 Skills Section

* List of technical skills
* Grouped by category

### 2.5 Projects Section

* Display multiple projects
* Each project must include:

  * Title
  * Description
  * Technologies used
  * GitHub link (if available)

### 2.6 Education Section

* HND completed
* Degree currently studying

### 2.7 Contact Section

* Phone number
* Email (future update)
* GitHub link
* LinkedIn (future update)

### 2.8 Resume Section

* Download button (file added later)

### 2.9 Footer

* Copyright
* Simple design

---

## 3. UI / Design Requirements

* Clean and modern design
* Fully responsive (mobile, tablet, desktop)
* Light and Dark mode support
* Easy to read typography
* Professional color scheme
* Smooth animations (not heavy)
* Fast loading

---

## 4. Technical Requirements

* Use only:

  * HTML
  * CSS
  * JavaScript

* No heavy frameworks (React, Angular, etc.)

* Code must be clean and well structured

* Use reusable CSS classes

* Separate files for HTML, CSS, JS

---

## 5. Performance Requirements

* Fast loading speed
* Optimized images
* Minimal unnecessary code
* Mobile friendly layout

---

## 6. Version Control

* Use Git for version control
* Push code to GitHub repository:
  https://github.com/Asterix7339

---

## 7. Deployment

* Host using Vercel
* Auto deploy from GitHub

## 8. Future Improvements

The following may be added later:

* Contact form backend
